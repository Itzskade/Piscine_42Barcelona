## üîë Permisos en Linux

Cada archivo tiene permisos divididos en tres grupos:

- **Propietario (user)**

- **Otros (others)**

| Permisos | Valor decimal | Significado            |
|----------|---------------|-----------------------|
| `---`    | 0             | Sin permiso           |
| `--x`    | 1             | Solo ejecutar         |
| `-w-`    | 2             | Solo escribir         |
| `-wx`    | 3             | Escribir y ejecutar   |
| `r--`    | 4             | Solo leer             |
| `r-x`    | 5             | Leer y ejecutar       |
| `rw-`    | 6             | Leer y escribir       |
| `rwx`    | 7             | Leer, escribir, ejecutar |

Para cambiar los permisos con `chmod` se usa un n√∫mero octal de 3 d√≠gitos, uno para cada grupo:


	- X es el permiso para el propietario

	- Z es el permiso para otros
